الوراثة — المفاهيم الأساسية

🏗️

مكونات الكلاس

كل كلاس يتألف من: fields (حقول)، properties (خصائص)، methods (دوال)، constructors (بواني).

🔑

Access Modifiers

public للجميع · private داخل الكلاس فقط · protected للكلاس والأبناء · internal داخل المشروع.

🧬

الوراثة Inheritance

كلاس جديد يرث الخصائص والدوال من كلاس آخر. يُقلّل التكرار ويُمثّل العلاقات الواقعية.

🔗

الكلمة المحجوزة base

تُستخدم في الكلاس الابن للوصول إلى constructor الأب أو دواله وخصائصه مباشرة.

مثال

وراثة بسيطة: Parent → Child

BasicInheritance.cs
class Parent { public string Name { get; set; } public int Age { get; set; } public Parent(string fName, int age) { Name = fName; Age = age; } } class Child : Parent { public int UniversityID { get; set; } // استدعاء constructor الأب عبر base public Child(string fName, int age, int uid) : base(fName, age) { UniversityID = uid; } public void PrintInfo() { Console.WriteLine("Name: " + Name); Console.WriteLine("Age: " + Age); Console.WriteLine("UID: " + UniversityID); } } // في Main: Child student = new Child("Ali", 20, 12345); student.PrintInfo();

محاضرة-02

Virtual & Override والتعددية

🔓

virtual

تكتب في الأب لتقول: "يمكن لأي وريث تعديل هذه الدالة." — الدالة لها تنفيذ افتراضي لكنه قابل للتغيير.

✏️

override

تكتب في الابن لتعيد تعريف دالة virtual من الأب. لا يمكن استخدامها بدون virtual في الأب.

🎭

Polymorphism

نفس المتغير · نفس الدالة · لكن النتيجة تختلف حسب نوع الكائن الحقيقي أثناء التنفيذ.

شروط

شروط Polymorphism

01

وجود وراثة بين الكلاسات

02

استخدام virtual في الأب

03

استخدام override في الابن

مثال

Animal → Dog / Cat

Polymorphism.cs
class Animal { public virtual void MakeSound() { Console.WriteLine("Animal Sound"); } } class Dog : Animal { public override void MakeSound() => Console.WriteLine("Bark"); } class Cat : Animal { public override void MakeSound() => Console.WriteLine("Meow"); } // في Main — نفس المتغير، نتيجة مختلفة: Animal a = new Dog(); a.MakeSound(); // Bark a = new Cat(); a.MakeSound(); // Meow

class Grandfather { public virtual void Greet() => Console.WriteLine("سلام من الجد"); } class Father : Grandfather { public override void Greet() => Console.WriteLine("سلام من الأب"); } class Son : Father { public override void Greet() // يمكن override مرة أخرى => Console.WriteLine("سلام من الابن"); } // Polymorphism — يُنفَّذ نسخة الابن دائماً: Grandfather f = new Father(); f.Greet(); // سلام من الأب Grandfather s = new Son(); s.Greet(); // سلام من الابن
محاضرة-03

abstract — الكلاس المجرد والدالة المجردة

🚫

لا object من abstract

لا يمكن إنشاء كائن مباشر من كلاس abstract. يُستخدم فقط كقالب للكلاسات الأبناء.

📋

دوال عادية + مجردة

يمكن أن يحتوي على دوال عادية (لها جسم) ودوال مجردة (بدون جسم {}).

⚡

إجبار التنفيذ

أي كلاس يرث من abstract يجب أن يطبّق جميع الدوال المجردة أو يكون هو أيضاً abstract.

قواعد

قواعد الدالة المجردة

01

لا يوجد جسم للدالة {}

02

تُجبر الكلاسات الأبناء على تنفيذها بالكلمة override

03

تُكتب داخل كلاس abstract فقط

أمثلة

أمثلة متعددة

Shape.cs
abstract class Shape { public abstract double CalculateArea(); // مجردة — بدون جسم } class Square : Shape { public double Side { get; set; } public Square(double side) { Side = side; } public override double CalculateArea() => Side * Side; // مساحة المربع } class Cube : Square { public Cube(double side) : base(side) { } public override double CalculateArea() => 6 * base.CalculateArea(); // المساحة الكلية للمكعب } // Main: double x = Convert.ToDouble(Console.ReadLine()); Shape s1 = new Square(x); Shape c2 = new Cube(x); Console.WriteLine("Square Area: " + s1.CalculateArea()); Console.WriteLine("Cube Surface Area: " + c2.CalculateArea());

محاضرة-04

Interface — الواجهة

Interface هو عقد (Contract) يحدد ما يجب أن يحتويه الكلاس من دوال وخصائص — بدون كتابة التنفيذ. كل الدوال فيه مجردة تلقائياً.
📝

لا متغيرات / لا بواني

الواجهة لا تحتوي fields عادية ولا constructors — فقط دوال وخصائص.

🔡

الاسم يبدأ بـ I

بالاتفاقية يبدأ اسم الواجهة بحرف I — مثل IEmployee, IShape.

🔀

تعدد الواجهات

كلاس واحد يمكنه تطبيق أكثر من واجهة — الحل البديل لتعدد الوراثة المحظور في C#.

وظيفة 1: Vehicle (Abstract Class)

السؤال: اكتب برنامج عرف فيه كلاس مجرد (abstract) مركبة Vehicle يحتوي اسم المركبة ودالة طباعة لطباعة اسم المركبة وسرعتها ودالة مجردة لحساب السرعة. وأنشئ Class Car يرث من Vehicle والزمن ويحسب سرعة المركبة حيث أن السرعة = المسافة ÷ الزمن. في Main أنشئ كائناً واطبع اسم المركبة وسرعتها.

Solution.cs
abstract class Vehicle { public string VehicleName { get; set; } public void Print() { Console.WriteLine($"المركبة: {VehicleName} السرعة: {CalculateSpeed()}"); } public abstract double CalculateSpeed(); } class Car : Vehicle { public double Distance { get; set; } public double Time { get; set; } public Car(string name, double d, double t) { VehicleName = name; Distance = d; Time = t; } public override double CalculateSpeed() { return Distance / Time; } } class Program { static void Main() { Vehicle v = new Car("Toyota", 300, 3); v.Print(); // المركبة: Toyota السرعة: 100 } }

وظيفة 2: Shape / Square / Cube (Abstract Class)

السؤال: اكتب برنامج عرف فيه كلاس مجرد (abstract) Shape يحتوي على دالة مجردة لحساب المساحة. وأنشئ Class Square يرث من Shape، وأنشئ Class Cube يرث من Square. في Main يدخل طول الضلع من المستخدم ويحسب مساحة المربع ومساحة المكعب ويطبعهما.

Solution.cs
abstract class Shape { public abstract double CalculateArea(); } class Square : Shape { public double Side { get; set; } public Square(double side) { Side = side; } public override double CalculateArea() { return Side * Side; } } class Cube : Square { public Cube(double side) : base(side) { } public override double CalculateArea() { return 6 * base.CalculateArea(); } } class Program { static void Main() { Console.WriteLine("Enter Side length?"); double x = Convert.ToDouble(Console.ReadLine()); Shape s1 = new Square(x); Shape c1 = new Cube(x); Console.WriteLine("Square Area: " + s1.CalculateArea()); Console.WriteLine("Cube Surface Area: " + c1.CalculateArea()); } }

وظيفة 3: BankAccount / SavingAccount / CurrentAccount

السؤال: اكتب برنامج عرف فيه كلاس مجرد (abstract) BankAccount يحتوي على اسم صاحب الحساب والرصيد ودالة مجردة لحساب الفائدة. SavingAccount يرث من BankAccount ويحسب الفائدة بنسبة 5%. CurrentAccount يرث من BankAccount ويحسب الفائدة بنسبة 2%. في Main أنشئ حسابين واطبع الفائدة لكل منهما.

السؤال: أنشئ واجهة ISpeed تفرض وجود دالة CalculateSpeed(). ثم أنشئ كلاساً مجرداً Vehicle يحتوي على Distance و Time. ثم أنشئ كلاسين Car و Bike يرثان من Vehicle ويطبقان الواجهة. في Main أنشئ كائنين سيارة ودراجة واطبع سرعة كل منهما.

وظيفة 5: IPrint + Media (Interface + Abstract)

السؤال: عرف واجهة IPrint تحتوي على دالة Print(). أنشئ كلاساً مجرداً Media يحتوي Title و Year. أنشئ Movie و Book يرثان من Media ويطبقان IPrint. في Main أنشئ قائمة List واطبع معلومات كل عنصر.

وظيفة 9: Circle / Cylinder (Inheritance)

السؤال: اكتب برنامج يستخدم الوراثة يقوم بتعريف Class Circle يحتوي نصف القطر ويحسب مساحة الدائرة. Cylinder يرث من Circle ويضيف الارتفاع ويحسب مساحة الأسطوانة. تابع باني لملئ الخصائص بالقيم.

Solution.cs
class Circle { public double Radius { get; set; } public Circle(double radius) { Radius = radius; } public double CalculateArea() { return Math.PI * Radius * Radius; } } class Cylinder : Circle { public double Height { get; set; } public Cylinder(double radius, double height) : base(radius) { Height = height; } public double CalculateArea() { return 2 * Math.PI * Radius * (Radius + Height); } } class Program { static void Main() { Cylinder c = new Cylinder(5, 10); Console.WriteLine("Circle Area: " + base.CalculateArea()); Console.WriteLine("Cylinder Area: " + c.CalculateArea()); } }

وظيفة 11: Point / Circle (Inheritance)

السؤال: اكتب برنامج يستخدم الوراثة يقوم بتعريف Class Point يحتوي إحداثيات (x,y) ويطبعها. Circle يرث من Point ويضيف نصف القطر ويحسب مساحة الدائرة ويطبع معلومات الدائرة. تابع باني لملئ الخصائص بالقيم.

class Point { public double X { get; set; } public double Y { get; set; } public Point(double x, double y) { X = x; Y = y; } public void Print() { Console.WriteLine("Point: ({0}, {1})", X, Y); } } class Circle : Point { public double Radius { get; set; } public Circle(double x, double y, double radius) : base(x, y) { Radius = radius; } public double CalculateArea() { return Math.PI * Radius * Radius; } public void PrintInfo() { Print(); Console.WriteLine("Radius: " + Radius); Console.WriteLine("Area: " + CalculateArea()); } } class Program { static void Main() { Circle c = new Circle(3, 4, 5); c.PrintInfo(); } }

وظيفة 12: Shipping / FastShipping (Virtual & Override)

السؤال: اكتب برنامج يستخدم الوراثة يقوم بتعريف Class Shipping يحتوي الوزن ويحسب تكلفة الشحن العادي. FastShipping يرث من Shipping ويضيف رسوم إضافية (5 + 10*كل كجم). تابع باني لملئ الخصائص بالقيم. دالة حساب التكلفة virtual.

Solution.cs
class Shipping { public double Weight { get; set; } public Shipping(double weight) { Weight = weight; } public virtual double CalculateCost() { return 5 * Weight; // 5 لكل كجم } } class FastShipping : Shipping { public FastShipping(double weight) : base(weight) { } public override double CalculateCost() { return base.CalculateCost() + 10 * Weight; // رسوم إضافية } } class Program { static void Main() { Shipping s1 = new Shipping(10); Shipping s2 = new FastShipping(10); Console.WriteLine("Normal Cost: " + s1.CalculateCost()); Console.WriteLine("Fast Cost: " + s2.CalculateCost()); } }

وظيفة 13: Circle / Cylinder (Virtual & Override)

السؤال: اكتب برنامج يستخدم الوراثة يقوم بتعريف Class Circle يحتوي نصف القطر ويحسب مساحة الدائرة. Cylinder يرث من Circle ويضيف الارتفاع ويحسب مساحة الأسطوانة. دالة حساب المساحة virtual. تابع باني لملئ الخصائص بالقيم.

Solution.cs
class Circle { public double Radius { get; set; } public Circle(double radius) { Radius = radius; } public virtual double CalculateArea() { return Math.PI * Radius * Radius; } } class Cylinder : Circle { public double Height { get; set; } public Cylinder(double radius, double height) : base(radius) { Height = height; } public override double CalculateArea() { return 2 * Math.PI * Radius * (Radius + Height); } } class Program { static void Main() { Circle c = new Circle(5); Cylinder cyl = new Cylinder(5, 10); Console.WriteLine("Circle Area: " + c.CalculateArea()); Console.WriteLine("Cylinder Area: " + cyl.CalculateArea()); } }

وظيفة 14: Employee / HourlyEmployee (Virtual & Override)

السؤال: اكتب برنامج يستخدم الوراثة يقوم بتعريف Class Employee يحتوي الاسم والراتب ودالة الزيادة (virtual). HourlyEmployee يرث منه ويحتوي عدد الساعات وسعر الساعة ويحسب الراتب مع الزيادة. تابع باني لملئ الخصائص بالقيم.

Solution.cs
class Employee { public string Name { get; set; } public double Salary { get; set; } public Employee(string name, double salary) { Name = name; Salary = salary; } public virtual double CalculateSalary() { return Salary; } } class HourlyEmployee : Employee { public int Hours { get; set; } public double HourRate { get; set; } public HourlyEmployee(string name, double salary, int hours, double rate) : base(name, salary) { Hours = hours; HourRate = rate; } public override double CalculateSalary() { return Salary + (Hours * HourRate); } } class Program { static void Main() { Employee e1 = new Employee("Ali", 5000); Employee e2 = new HourlyEmployee("Sara", 3000, 40, 50); Console.WriteLine("Employee Salary: " + e1.CalculateSalary()); Console.WriteLine("Hourly Salary: " + e2.CalculateSalary()); } }
